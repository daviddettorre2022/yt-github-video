Github

Push du contenu sur le repository de github

initialiser une seule fois avec git init

ensuite 3 commandes sont necessaires :

git add . (cette commmande push tout le dossier)

ou bien utiliser git index.html (cette commmande uniquement le fichier index.html)

puis : git commit -m 'explication du procédé' (ce qui est entre parenthese est le message qui explique ce que l'on vient de commit)

puis enfin : git push
